### Added

- CLI with `lint` and `version` commands using cobra ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- JSON Schema parser for linting (supports `$defs`, `definitions`, `anyOf`, `oneOf`) ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- Union detection with discriminator field analysis ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- Nullable pattern detection (`anyOf: [T, null]`) ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- Reference pattern detection (`anyOf: [ComponentReference, BaseXxx]`) ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- Configurable discriminator field priority (`component_type`, `type`, `kind`) ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- Multiple output formats: text (default), JSON, GitHub Actions annotations ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
- Exit codes: 0 (ok), 1 (errors), 2 (warnings only) ([`f8ed488`](https://github.com/grokify/schemago/commit/f8ed488))
