## [v0.2.0] - 2026-02-08

### Added

- `--profile` / `-p` flag to select linting profile (`default`, `scale`) ([`c0fed40`](https://github.com/grokify/schemalint/commit/c0fed40))
- Scale profile with `composition-disallowed` rule: disallow `anyOf`, `oneOf`, `allOf` ([`c0fed40`](https://github.com/grokify/schemalint/commit/c0fed40))
- Scale profile with `additional-properties-disallowed` rule: disallow `additionalProperties: true` ([`c0fed40`](https://github.com/grokify/schemalint/commit/c0fed40))
- Scale profile with `missing-type` rule: require explicit `type` field ([`c0fed40`](https://github.com/grokify/schemalint/commit/c0fed40))
- Scale profile with `mixed-type-disallowed` rule: disallow type arrays like `["string", "number"]` ([`c0fed40`](https://github.com/grokify/schemalint/commit/c0fed40))
- Goreleaser configuration for cross-platform builds and Homebrew tap ([`b5d1fab`](https://github.com/grokify/schemalint/commit/b5d1fab))

### Changed

- **BREAKING**: Rename binary from `schemago` to `schemalint` ([`a96efd5`](https://github.com/grokify/schemalint/commit/a96efd5))
- **BREAKING**: Rename module from `github.com/grokify/schemago` to `github.com/grokify/schemalint` ([`a96efd5`](https://github.com/grokify/schemalint/commit/a96efd5))
- Update CLI description to emphasize static type compatibility for all languages (Go, Rust, TypeScript, etc.) ([`a96efd5`](https://github.com/grokify/schemalint/commit/a96efd5))

## [v0.1.0] - 2026-01-17

### Added

- CLI with `lint` and `version` commands using cobra ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- JSON Schema parser for linting (supports `$defs`, `definitions`, `anyOf`, `oneOf`) ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- Union detection with discriminator field analysis ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- Nullable pattern detection (`anyOf: [T, null]`) ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- Reference pattern detection (`anyOf: [ComponentReference, BaseXxx]`) ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- Configurable discriminator field priority (`component_type`, `type`, `kind`) ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- Multiple output formats: text (default), JSON, GitHub Actions annotations ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
- Exit codes: 0 (ok), 1 (errors), 2 (warnings only) ([`f8ed488`](https://github.com/grokify/schemalint/commit/f8ed488))
